#### mac查看端口、进程号、进程名的命令

薛延祥
0.086
2019.10.08 10:42:30
字数 124
阅读 675
1、查看进程号

ps -ef | grep 进程名

2、查看端口被哪个进程监听

sudo lsof -i :端口

3、查看进程监听的端口

sudo lsof -nP -p 进程号 | grep LISTEN

sudo lsof -nP | grep LISTEN | grep 进程号

4、查看监听端口的进程

sudo lsof -nP | grep LISTEN | grep 端口号

————————————————

示例：

查看3306端口号被占用情况

sudo lsof -nP | grep LISTEN 3306
